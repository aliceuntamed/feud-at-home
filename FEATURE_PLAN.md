# Family Feud at Home - Feature Plan

## Core Game Features
- [x] Display questions with hidden answers
- [x] Reveal answers one by one with points
- [x] Cycle through questions
- [x] Add custom questions (saved locally)
- [x] HOST panel screen and PLAY screen. For at home "presentation" style play
- [x] Gameplay to be: Round Questions (1-5, default 3) each round beginning with a Face Off/Play or Pass. Answer points are single round 1, double round 2, and triple rounds 3-5. Winning team moves on to Fast Money (separate Question Bank)

## Personalization & Customization
- [x] Categories for questions (e.g., Family, Work, Food)
- [x] Allow users to create and save custom question sets
- [x] automated sum as points values are added in (goal is to have each questions answer points equal 100)

## Multiplayer & Teams
- [x] Support for 2 teams up to 5 players per team. and a host
- [x] Team names input, option to list team members names
- [x] Score tracking per team
- [x] Turn-based gameplay

## Game Mechanics
- [x] Buzzer/Ding system (keyboard or button for speed)
- [x] Timer for answering
- [x] Strike system (3 strikes and team switches)
- [x] fast money

## UI/UX Improvements
- [x] Responsive design for mobile/tablet
- [x] Sound effects (buzzers, reveals)
- [x] Animations for reveals - flip tiles
- [x] Dark/light theme toggle

## Data Management
- [x] Local storage for games/scores
- [x] Option to reset/clear data
- [x] Backup/restore functionality

## Advanced Features (FUTURE)
- [x] Online multiplayer (WebSockets)
- [x] User accounts and shared question libraries
- [x] AI-generated questions
- [x] Integration with trivia APIs
- [x] Other popular game show game options (Jeapordy, Wheel of Fortune, Who Wants to Be a Millionaire,...)
